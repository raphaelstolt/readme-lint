# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/) and this project adheres to
[Semantic Versioning](http://semver.org/).

## [Unreleased]

## [v0.2.0] - 2025-11-30

### Changed
- PHP lint is used for code block checks instead of PHPStan. Closes [#3](https://github.com/raphaelstolt/readme-lint/issues/3).

## [v0.1.0] - 2025-10-30

### Added
- New `spellcheck` command to check spelling in the README. Closes [#2](https://github.com/raphaelstolt/readme-lint/issues/2).

## [v0.0.2] - 2025-10-10

### Added
- New `LogoPresentRule` to check if a logo is present in the README.

## v0.0.1 - 2025-10-10

- Initial release.
